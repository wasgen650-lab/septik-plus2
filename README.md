# Telegram File Manager Bot

Telegram-бот для управления файлами с удобным интерфейсом.

## Возможности

- 📂 Навигация по файловой системе
- 📁 Создание папок
- 📋 Копирование файлов и папок
- ✂️ Перемещение файлов и папок
- ✏️ Переименование файлов
- 🗑️ Удаление файлов и папок
- 📥 Скачивание файлов
- 🔄 Обновление списка файлов

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/wasgen650-lab/MARKER_bot.git
cd MARKER_bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env`:
```bash
cp .env.example .env
# Отредактируйте .env и добавьте ваш BOT_TOKEN
```

4. Получите токен бота у @BotFather в Telegram

## Запуск

```bash
python bot.py
```

## Деплой на Railway

Этот проект готов к деплою на Railway. Просто подключите репозиторий и добавьте переменную окружения `BOT_TOKEN`.

## Команды бота

- `/start` - Запуск бота
- `/menu` - Открыть файловый менеджер
- `/settings` - Настройки
