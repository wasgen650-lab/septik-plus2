# Септик-Плюс

Лендинг для компании "Септик-Плюс" — ассенизаторские услуги в Краснодаре.

## Возможности

- 🎨 **3D интро-анимация** — сфера из 15,000 частиц (Three.js)
- 🧮 **Калькулятор услуг** — автоматический расчёт стоимости
- 📱 **Telegram-бот** — отправка заявок напрямую в Telegram
- 📊 **Яндекс.Метрика** — аналитика посещений
- 📱 **Адаптивный дизайн** — корректное отображение на мобильных устройствах

## Технологии

- PHP (обработка форм)
- Three.js (3D анимации)
- CSS Grid/Flexbox (вёрстка)
- Telegram Bot API

## Установка

1. Загрузите файлы на PHP-хостинг
2. Настройте Telegram-бота:
   - Создайте бота через [@BotFather](https://t.me/BotFather)
   - Получите токен
   - Укажите токен и chat_id в `index.php`

## Настройка Telegram-бота

Откройте `index.php` и измените:

```php
$tg_bot_token = 'ВАШ_ТОКЕН_БОТА';
$tg_chat_ids  = ['ВАШ_CHAT_ID_1', 'ВАШ_CHAT_ID_2'];
```
